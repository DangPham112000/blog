# Hugo quick start

### create new site

```
hugo new site [site-name]
```

### create new blog

```
hugo new content posts/[post-name]
```

### first come
- Install `golang`,`dart-sass`,`hugo`
- Clone theme:
    - Go to `themes` folder: `cd engineerblog/themes`
    - Clone theme: `git clone `

### run dev

```
hugo server
```

### build hugo

```
hugo -t [theme-name]
hugo -t hugo-book
```
